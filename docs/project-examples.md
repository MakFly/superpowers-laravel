# Project Examples (Any Domain)

Use these examples as memory anchors. Pick the closest complexity tier.

## Simple
**Example**: Basic CRUD API.
- Routes: index/show/store/update
- Data: migrations + Eloquent models
- Goal: clean resources, simple validation

## Medium
**Example**: SaaS with auth + billing.
- Routes: `/login`, `/app`, `/settings`
- Data: policies + resources + tests
- Goal: secure access, stable DX

## Complex
**Example**: Event-driven platform.
- Routes: multiple domains + admin
- Data: queues/jobs, cache tags, observability
- Goal: scale, async processing, reliability
